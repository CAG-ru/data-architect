О приложении

Data Architect – мощный инструмент для самостоятельного создания витрин данных.

Простой веб-интерфейс
Вы сможете создать ваши первые рабочие «витрины» в первый же день знакомства с системой
  • Самостоятельно создавайте витрину из предоставленных наборов данных
  • Вам больше не нужно ждать создания витрины специалистами ИТ
  • Подключайтесь к любой готовой витрине
Автоматизация
  • После выбора необходимых параметров для создания витрины, процесс идет автоматически
  • Вам придет оповещение о создании витрины и параметры подключения к ней
  • Обновление витрин происходит также в автоматическом режиме при изменении исходных данных
Экономия ресурсов
  • Если витрина больше не нужна – она может быть удалена или по истечении срока действия или по вашему желанию

Основные разделы приложения:

Витрины данных

Создание новых витрин данных. Работа с витринами данных.

Источники данных

Просмотр и редактирование метаданных для подключенных источников данных.

Связи между источниками

Создание и редактирование связей между источниками данных.

Подключение к источникам данных

Настройка соединений с источниками данных.
